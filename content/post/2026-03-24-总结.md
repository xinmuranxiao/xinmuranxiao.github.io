# 跨文件(不引用头文件实现函数跨命名空间类的调用)
``` cpp {lineNumbers="true"}
//使用using定义一个函数指针变量(以输出函数为例)
using Out=void*(*)(std::string);
//------函数1.cpp----
Out put=out;
//------函数2.cpp----
void out(std::string){
    cout<<std::string<<endl;
}
```
- 这样就可以实现文件的不引用调用函数了
