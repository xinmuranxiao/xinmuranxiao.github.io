# 一、算法
## (1)gcd(最大公因数)
### 1.while循环
``` cpp {lineNumbers="true"}
template<class T>
T gcd(T i,T j){
    while(i!=0){//使用辗转相除
        int temp=j%i;
        j=i;
        i=temp;
    }
    return j;
}
```
### 2.递归
``` cpp {lineNumbers="true"}
template<class T>
T gcd(T a, T b) {
    return b == 0 ? a : gcd(b, a % b);
}
```
## (2)使用pair存储分数中重载运算符(需使用gcd函数(cpp14以上好像支出原生gcd))
### +：
``` cpp {lineNumbers="true"}
template<class T>
pair<T,T> operator+(pair<T,T> a,pair<T,T> b){
    T g=gcd(a.first*b.second+a.second*b.first,a.second*b.second);
    pair<T,T>f={(a.first*b.second+a.second*b.first)/g,a.second*b.second/g};
    return f;
}
```
### *：
``` cpp {lineNumbers="true"}
template<class T>
pair<T,T> operator*(pair<T,T> a,pair<T,T> b){
    T g=gcd(a.first*b.first,a.second*b.second);
    pair<T,T>f={a.first*b.first/g,a.second*b.second/g};
    return f;
}
```
